# 🤖 AI Chatbot Website

## 📌 Overview
This project is a simple and interactive AI Chatbot web application built using HTML, CSS, and JavaScript. It allows users to communicate through a chat interface and receive dynamic responses, simulating a basic conversational AI system.

---

## 🚀 Features
- 💬 Interactive chat interface
- ⚡ Real-time message responses
- 🤖 Basic natural language handling
- 📜 Chat history display
- 🎨 Clean and user-friendly UI
